### Description
This is my solution for the **Minimal Blog Card** challenge from DevChallenges.io. I created a clean card component that displays an article preview with an image, category tag, title, and author footer.

### Built With
- Semantic HTML5 markup
- CSS3
- Flexbox for centering
- Google Fonts (Sora)

### What I Learned
In this project, I focused on:
- Importing and using custom fonts (Sora) from Google Fonts.
- Styling elements with specific `box-shadow` to create depth.
- Using `display: inline-block` to create pill-shaped tags.
- Structuring a card layout with proper padding and margins.
